CSCE 242 Web Applications

Aiden Mckay

CSCE is a class teaching students how to design an effective website from the front end, using

sample websites as a practice method. The second goal of the class focuses on the javascript

aspect of web design, which brings said website more applications and functions. The last part

of the class focuses on integrating the learned javascript into recieving and storing data.
